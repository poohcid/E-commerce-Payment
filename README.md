# E-commerce-Payment
Payment Module คำนวนการสั่งซื้อสินค้า เพิ่มโปรโมชันในการซื้อสินค้า และระบบเติมเงิน

## Product Service
  - มี Wallet (เหมือนบัญชีการเงิน)
  - เติมเงินเข้า Wallet
  - บันทึกรายการสั่งซื้อ
  - เพิ่ม Promotion ในการจ่ายเงินได้
  - มีการ Refund ได้
  - บันทึกเงินเข้าเงินออก
